## Mandelbrot Rust program
----
##### Usage
- Install<br>
git clone https://github.com/tenseurevyeux/mandelbrot.git<br>
- Build(rustc && cargo require)<br>
cd mandelbrot && cargo build --release<br>
- Use<br>
./target/release/mandelbrot --help<br>
